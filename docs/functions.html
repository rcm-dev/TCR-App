Documentation - Functions

Sidebar

The sidebar is has built in functionality so that it can be turned on and off by the user and the preference is remembered in a cookie. The sidebar can also be configured to the left or the right by changing the class on the body to either sidebar-left or sidebar-right. Some clever CSS handles the rest.

Tooltips

Admin Base uses the excellent tipsy tooltips plug-in to show tooltips. I've built a couple of inbuilt classes to help you get the most configurability out of the plugin. To add a tooltip to something simply add the class "tooltip" and add the tooltip text in the title attribute of the tag. So take this for example:



You can configure how the tooltip shows more extensively by adding polar directions to class. So for example adding the class " tipsy-north" will make your tooltip appear above the object instead of the default below.

Tooltips can also be used on forms to add a hint for users.

Charts

Admin Base uses the Google visualizations library (http://code.google.com/apis/charttools/index.html) to visualize data. It's quite simple to setup and customize. You can see an example on the dashboard preview. I've also included a bit of JavaScript that resizes the chart if you change the width of the browser (or use the dynamic width feature of the template).
