Documentation - CSS Elements

Generic

This template has several built in classes for easy formatting.

.cl
A clearing class to clear floats without taking up vertical space
.fl
float something left
.fr
float something right
.fn
specifically don't float something
.tl
align text left
.tr
align text right
.tc
align text center
.hd
hide an object
.strong
bold text
.margin-left, .margin-right, .margin-top, .margin-down
add a 20px margin
.border-left, . border-right, . border-top, . border-down
add a border with a 10px margin
.txt-small
font size to 85%
.txt-smaller
font size to 73%
.txt-light
font lighter
.txt-lighter
font even lighter

Calendar

Admin Base includes three different calendar styles. Mini, Medium and Full. They all use a common table structure.

Mini Calendar

The Mini Calendar is 200px in width. It can be position left or right using the built in float classes. (.fl and .fr)
